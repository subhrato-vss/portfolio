# Subhrato Portfolio

A modern, responsive personal portfolio website built with **React** and **Vite**, designed to showcase my skills, projects, and experience as a developer.

The project uses **Material UI** for a clean and accessible design system, along with **Framer Motion** to deliver smooth, engaging animations and transitions. Built with performance and scalability in mind, this portfolio serves as a central place to highlight my work and online presence.

## Tech Stack

- **React 19**
- **Vite**
- **Material UI (MUI)**
- **Framer Motion**
- **Emotion (CSS-in-JS)**

## Features

- Responsive and modern UI
- Smooth animations and page transitions
- Fast development and build times with Vite
- Component-based architecture
